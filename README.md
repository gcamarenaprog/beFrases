# WordPress Plugin beFrases

A simple plugin for Wordpress to manage, save, register and publish your quotes of your favorite authors.

## Description

A simple and easy to use plugin for Wordpress that will allow you to manage a list of quotes with their categories and
authors, and you will also be able to publish in the sidebar with a widget with the category filter option.

## Features

- Easy to visualize data presentation.
- Manage your famous quotes easily visible on the dashboard through a table.

- Option to alphabetical order in author, category and quote fields.
- Quote search filter option, just type author, category or any word or part of the quote.
- Select an author from a previously saved list of authors for each quote.
- Select a category from a previously saved list of authors for each quote.
- See, add, update or delete any quote in the same screen.

- Option to alphabetical order in name, description or number of quotes fields.
- Category search filter option, just type name, description or any word or part of the category information.
- See, add, update or delete any category in the same screen.

- Option to alphabetical order in name or number of quotes fields.
- Author search filter option, just type name or any word or part of the author information.
- See, add, update or delete any author in the same screen.

- Option to select text quote style (normal, italic, bold or italic and bold).
- Option to select text quote alignment (right, left, center o justify).
- Option to select text author style (normal, italic, bold or italic and bold).
- Option to select text author alignment (right, left, center o justify).
- Demo function, in "settings" section that allows you to preview texts widget in front-end.
- 
- Widget to random display your famous quotes anywhere on your site.
- Select a category to post in the widget.

## Installation

Installing "beFrases" using the following steps:

1. Download the plugin via https://guillermocamarena.com/befrases
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress

## Frequently Asked Questions

### How I search words, category name or author in the quotes list?

Go to option menu: 'Manage' -> 'Search filter', just type the word and the matches will be filtered.

### How to add a quote?

Go to option menu: 'Manage' -> 'Add quote'
Note: remember to omit the quotation marks at the beginning and end of the quote.

### How to edit a quote?

Go to option menu: 'Manage' -> 'Quote list' -> 'Click on edit button'

### How to edit a quote?

Go to option menu: 'Manage' -> 'Quote list' -> 'Click on edit button'

### How to delete a quote?

Go to option menu: 'Manage' -> 'Quote list' -> 'Click on delete button'

### How to edit a category?

Go to option menu: 'Categories' -> 'Quote list' -> 'Click on edit button'

### How to edit a category?

Go to option menu: 'Categories' -> 'Quote list' -> 'Click on edit button'

### How to delete a category?

Go to option menu: 'Categories' -> 'Quote list' -> 'Click on delete button'

### How to edit an authors?

Go to option menu: 'Authors' -> 'Quote list' -> 'Click on edit button'

### How to edit an authors?

Go to option menu: 'Authors' -> 'Quote list' -> 'Click on edit button'

### How to delete an authors?

Go to option menu: 'Authors' -> 'Quote list' -> 'Click on delete button'

### How to select text alignment for quote?

Go to option menu: 'Settings' -> 'Quote alignment' -> 'Select: center, right, justify or left.'
Note: You can see a demo of the text in the "Demo" section.

### How to select text style for quote?

Go to option menu: 'Settings' -> 'Quote style' -> 'Select: normal, italics, bold or italics and bold.'
Note: You can see a demo of the text in the "Demo" section.

### How to select text alignment for author?

Go to option menu: 'Settings' -> 'Quote alignment' -> 'Select: center, right, justify or left.'
Note: You can see a demo of the text in the "Demo" section.

### How to select text style for author?

Go to option menu: 'Settings' -> 'Quote style' -> 'Select: normal, italics, bold or italics and bold.'
Note: You can see a demo of the text in the "Demo" section.

Note that if you include your own classes, or third-party libraries, there are three locations in which said files may
