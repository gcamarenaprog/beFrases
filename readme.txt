== Description ==

"beFrases" a simple and easy-to-use plugin that will allow you to manage a list of phrases and authors and display in your sidebar (widget).

**Plugin Features**

- Famous phrases easily visible on the dashboard through a table.
- Select category for each phrase.
- Option to alphabetical order in author and phrase fields.
- Option to filter search words in author text or phrase text.
- Easy to visualize data presentation.
- Option to select text styles personalized.
- Option text align and font style for author and phrase.
- Widget to random display your famous phrases anywhere on your site.

Want to contribute? [Fork the GitHub repository] (https://github.com/gcamarenaprog/befrases).

* This plugin was initially created for my website (https://bibliotecaenigmas.com)*


== Installation ==

Installing "beFrases" using the following steps:

1. Download the plugin via https://gcamarenaprog.com/befrases
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= When I order posts by hit count in the dashboard it doesn't show all of the posts - what gives? =
When you order posts by hit count in the post list table in the dashboard it will not show posts that have 0 hits. This is due to a limitation in WordPress itself, which will (most likely) be fixed in future versions. Once WordPress is able to accommodate this then I will update the plugin to work accordingly

= How I search words in the phrases? =
On the main plugin screen, there is a "search" filter textbox - just type the word and the matches will be filtered.

= How I search author name? =
On the main plugin screen, there is a "search" filter textbox - just type the name and the matches will be filtered. You can also make use of the custom filter that is in list mode under the author column.

= How I search cateogory name? =
On the main plugin screen, there is a "search" filter textbox - just type the name and the matches will be filtered. You can also make use of the custom filter that is in list mode under the category column.

= How do I all delete the data beFrases? =
On the main screen of the plugin, there is a button "Delete records" just click on the button, confirm and all data will be deleted.

= How can I edit or delete a phrase? =
On the main screen of the plugin, in the data table, each record has two buttons "Delete" and "Edit", just click on them.

? How can I add a phrase? =
On the main screen of the plugin, there is a button "Add phrase", just click and add the data of the new phrase.

For more information, you can see "Help Section" of the plugin on the admin.
